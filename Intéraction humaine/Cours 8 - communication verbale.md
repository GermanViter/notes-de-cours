# Les pieges a eviter
## Se méfier des ambiguités
- Mots a valeur **relative**
	- ex: *un peu* d'argent pour jeff Bezos ou Bill Gates
	- ex: "Peux -tu me remettre ça tantôt"
- **Ignorer les regles** pragmatiques
	- ex: "Les pompiers n'y on vu que du feux"
	- ex: "Vous êtes bien jolie ajd"
- Ne pas conciderrer les **connotations**
	- le choix de vocabulaire sou-entends toujours qq chose:
		- Sceptique *vs* Complotiste
- Se méfier des **mais**
	- Enleve de l'importance de l'idee d'avant
		- "T'es un bon employes, mais tu n'a pas de tacte avec les clients"
	- Idealement faire les deux idees séparés
	- Si l'importance est dans le **deuxieme** partie alors c pas grave
### Se méfier des fausses qstions
- Permet d'etre mois direct
- Reste souvent ambigue
### Se méfier des énonces impersonnels
- On est pas dans la peau de tt le monde
### Se méfier de complexites
- Phrases **longues**
- Idees **compliques**
- Termes **inventés**
### Se méfier des **formulations a la deuxieme personne**
- est tres souvent interprete comme accusateur
